# Roadmap

## Objetivo

Guiar a evolução do SDK sem perder a compatibilidade retro.

## Concluído

- visão do SDK
- organização em módulos
- escopo de plataformas alvo
- escopo de linguagens alvo

## Em andamento conceitual

- definição de `librn`
- definição de `librnui`
- definição de `librnaudio`
- definição de `rn-tsr`
- documentação de capabilities
- documentação de toolchains

## Próximos

- cliente CLI simples
- cliente TUI
- cliente gráfico VGA
- TSR para polling e notificações
- cliente Win3.x
- cliente Win9x
- integração QWK
- ponte IPX/IPXBox
- exemplos para Pascal e C++

## Status

Este roadmap é inicial e serve como orientação para contribuições.

## Uso futuro

O roadmap deve orientar prioridades sem transformar o SDK em uma pilha pesada.

## Próximos passos

- decompor entregas pequenas
- marcar o que é compatível com DOS puro
- marcar o que depende de emulador ou hardware real
