# Clientes

## Objetivo

Documentar os tipos de cliente previstos para o ecossistema RetroNet.

## Clientes previstos

- cliente CLI simples
- cliente TUI em modo texto
- cliente gráfico VGA
- TSR para polling e notificações
- futuro cliente Win3.x
- futuro cliente Win9x

## Status

São metas de produto e ecossistema, não entregas prontas.

## Uso futuro

- CLI para automação e integração
- TUI para ambientes de texto e DOS
- VGA para interfaces mais ricas sem sair do retro
- TSR para notificações leves

- separar responsabilidades de UI e transporte
