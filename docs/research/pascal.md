# Pascal

## Objetivo

Registrar a importância de Pascal no ecossistema RetroNet.

## Status

Pascal é uma linguagem relevante porque há muitos programadores retro
com experiência em Turbo Pascal, Borland Pascal e Delphi inicial.

Isso amplia a base de colaboradores e facilita a criação de clientes e
ferramentas para DOS e ambientes antigos.

## Uso futuro

- bindings para o SDK
- exemplos de cliente
- ferramentas e utilitários para DOS

## Próximos passos

- mapear compatibilidade com Turbo Pascal e Borland Pascal
- avaliar FreePascal e Delphi quando fizer sentido
