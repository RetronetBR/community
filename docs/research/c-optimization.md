# Otimização de C

## Objetivo

Registrar a trilha de pesquisa para escrever C pequeno, rápido e compatível com hardware limitado.

## Status

C continua sendo a linguagem base mais útil para núcleo, clientes e bibliotecas do SDK.

## Uso futuro

- reduzir memória
- reduzir chamadas desnecessárias
- melhorar desempenho de TSRs e clientes leves

## Próximos passos

- documentar padrões de alocação e buffer
- documentar estratégias de compilação para toolchains antigas
